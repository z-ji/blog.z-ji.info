Wordpress Posting in C# using xmlrpc api

First of all, you should enable XML-RPC call on wordpress if you use wordpress 2.6 and later.
How to enable XML-RPC? Just log in to your blog with the admin account, and go to Settings, and then Writing.
Check the box next to “XML-RPC” on the Writing options page, and save the settings.
You can directly consume XML-RPC calls in C#, or you can use some wrappers out there. The most popular wrapper out there is JoeBlogs. The source code is on github: https://github.com/alexjamesbrown/JoeBlogs. Once you get the source and make a successful build, you are ready to post in less then 10 lines of code.
